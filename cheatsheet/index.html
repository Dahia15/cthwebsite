Cheatsheet

Eerste stap :


We gaan er vanuit dat je dit doet op een PC of laptop met windows, om te beginnen moet je op de Windows knop drukken en daarna typ je in het zoek vak "cmd" hierdoor open je de terminal.

Tweede stap:


Ga naar je Pvaardig map via de terminal door middel van de cd command.

Derde stap:


Typ "Git status" om te zien of er geenn repository in de geopenede map zit.

Vierde stap:


Typ daarna "Git init" en daarna nog is "Git status" om te kijken hoe het nu zit in je repository, je zou nu moeten zien dat er nog geen bestanden of commits zijn en dat gaan we nu veranderen.

Vijfde stap:


Typ in de command line "Notepad" en er zal een notepad file verschijnen je kan dit bestand noemen wat je wilt zoals test.txt.

Zesde stap:


Wanneer je nu "Git status" doet zul je zien dat je test bestand nu verschijnt, om hem toe te voegen aan de repository doe je de command "Git add test.txt" en daarna kan je hem committen met de command "Git commit -m "First commit" deze command zorgt ervoor dat alle toegevoegde bestanden worden gecommit.

Zevende stap:


Nu ga je naar je browser en naar je github account en maak een nieuwe repository aan. Volg de instructies op het scherm en link je eerdere map aan de repository met de "Git remote add" command, hierna kan je "Git push" doen om je bestanden in de Github repository te krijgen.
